The home page for normal users. Basic home page elements are present.

About page for normal users. Contains description of the CSSA.

Events page for normal users. Names of events are listed in list and calendars. If a user clicks on an event in the calendar or list, additional details and a description will be displayed.

Client sponsors page for normal users. Information about CSSA sponsors will be listed here.

Home page for admins once logged in.

Attendance tool for admins. Clicking on an event in the calendar will bring up an attendance list. Users can check members off member attendance online during the event.

Event Scheduling tool for admins. Clicking on a date on the calendar brings up a list of events for that date. New events can be added, descriptions of existing events can be changed, and attendance lists for events can be managed.

Fund Management tool for admins. Fund changes and donations can be manually added online and will be stored in a list format.
